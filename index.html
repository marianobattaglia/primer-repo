Desafío clase N°6

Crear repositorio de Git

Formato: sube a la plataforma el screenshot del log de la rama master.Debe tener el nombre “Idea+Apellido”.

Sugerencia: luego de que veas que estás bien con esto, ¡pon Git en tu proyecto!

>> Consigna: crea un repositorio de Git.

>>Aspectos a incluir en el entregable:

  • Crea una carpeta e inicia Git.
  • Agrégale algún HTML, lo que quieras (imágenes, estilos, etcétera). Practica el uso de los 3 estados con la terminal.
  • Luego crea una rama.
  • Vuelve hacia un commit anterior.
  • Mergea la nueva rama con master.
